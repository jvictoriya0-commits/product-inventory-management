Product Inventory Management System - Backend API

A secure and scalable RESTful API for managing product inventory with JWT-based authentication. Built with Node.js, Express.js, and MongoDB.

Features

User Authentication - Register and login with JWT tokens
Product Management - Create, read, update, and delete products
User-specific Data - Each user can only access their own products
Search Functionality - Search products by name
Secure Routes - Protected endpoints with JWT verification
Input Validation - Data validation and error handling
